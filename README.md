# ClickHouse Skills for Claude Code

Build ClickHouse tables with sub-second queries, 10x compression, and zero full table scans.

## Installation

```bash
npx skills add https://github.com/obsessiondb/clickhouse-skills
```

## Available Skills

| Skill                             | Goals                                                              |
| --------------------------------- | ------------------------------------------------------------------ |
| **clickhouse-schema-design**      | Sub-second queries on billion rows, 10x compression, auto-retention |
| **clickhouse-query-optimization** | 10-100x faster queries, 90% less memory, eliminate full scans       |

## Roadmap

| Skill                         | Status  | Goals                                               |
